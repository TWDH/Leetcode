# 01.二分法模板

## 1.模板

```java
class Solution {
    public int search(int[] nums, int target) {
        //边界情况
        if(nums == null || nums.length == 0){
            return -1;
        }

        //头尾指针
        int start = 0;
        int end = nums.length - 1;
        //如果start / end相邻，则直接跳出while循环并判断
        while(start + 1 < end){
            int mid = start + (end - start) / 2;
            if(nums[mid] == target){
                return mid;
            }
            else if(nums[mid] < target){
                start = mid;
            }
            else{
                end = mid;
            }
        }

        if(nums[start] == target){
            return start;
        }
        if(nums[end] == target){
            return end;
        }

        return -1;
    }
}
```
