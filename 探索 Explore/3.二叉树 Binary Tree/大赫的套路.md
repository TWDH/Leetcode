# 树的遍历

**迭代法**：使用stack控制TreeNode节点

* 根：`res.add(node.val)`  
* 左：`node = node.left`
* 右：`node = node.right`
* 控制出栈入栈：
  * `stack.push(node.val)`
  * `stack.pop()`

整体代码逻辑

```java
while(!stack.isEmpty() || node != null){
    while(node != null){
        ...
        stack.push(node.val);
    }
    ...
    stack.pop();
}
```
