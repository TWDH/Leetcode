## 方法一：迭代法(Iterative)

```java
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        //1.初始化res存储答案
        List<Integer> res = new LinkedList<>();
        //2.特殊情况
        if(root == null){
            return res;
        }
        //3.建立stack模拟递归
        Deque<TreeNode> stack = new LinkedList<>();
        //建立pre用来记录已经遍历过的点
        TreeNode pre = null;
        
        while(!stack.isEmpty() || root != null){
            //左：先一直向左到底
            while(root != null){
                stack.push(root);
                root = root.left;
            }
            //出栈
            root = stack.pop();
            
            //右子树如果为null或者右子树已经遍历过，则只添加当前节点
            if(root.right == null || root.right == pre){
                //根
                res.add(root.val);
                //当前节点已经遍历完
                pre = root;
                root = null;
            }
            //右子树有节点
            else{
                //再次将当前点入栈
                stack.push(root);
                //右
                root = root.right;
            }    
        }
        return res;
    }
}
```

## 方法二：递归(recursive)

```java
class Solution {
    //1.初始化List储存答案
    List<Integer> res = new ArrayList<>();
    
    public List<Integer> postorderTraversal(TreeNode root) {
        postOrder(root);
        return res;
    }
    
    public void postOrder(TreeNode root){
        if(root == null){
            return;
        }
        //左
        postOrder(root.left);
        //右
        postOrder(root.right);
        //根
        res.add(root.val);
    }
}
```
