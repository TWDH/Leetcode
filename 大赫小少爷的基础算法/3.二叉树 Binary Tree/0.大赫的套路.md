# 树的遍历

**迭代法**：使用stack控制TreeNode节点

* 根：`res.add(node.val)`  
* 左：`node = node.left`
* 右：`node = node.right`
* 控制出栈入栈：
  * `stack.push(node.val)`
  * `stack.pop()`

整体代码逻辑

```java
while(!stack.isEmpty() || node != null){
    while(node != null){
        ...
        stack.push(node.val);
    }
    ...
    stack.pop();
}
```

### ArrayDeque和LinkedList区别

**ArrayDeque:**

* 数组结构
* 插入元素不能为null
* 无法确定容量时，后期扩容收到影响

**LinkedList**

* 链表结构
* 插入元素可以为null
* 无法确定数据容量，有更好表现
