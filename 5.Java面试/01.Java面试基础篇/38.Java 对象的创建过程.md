### Step1:类加载检查

- 虚拟机遇到一条 new 指令时，首先将去检查这个指令的参数是否能在 **常量池** 中定位到这个 **类的符号引用** ，并且检查这个符号引用代表的类是否已被加载过、解析和初始化过。如果没有，那必须先执行相应的 **类加载 **过程。
